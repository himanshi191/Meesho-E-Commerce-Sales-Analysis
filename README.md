## 📌 Project Overview

This project performs **end-to-end exploratory data analysis (EDA)** on a real-world e-commerce dataset similar to platforms like **Meesho, Amazon, and Flipkart**.

The objective of this project is to analyze **customer behavior, product performance, seller contribution, and sales trends** in order to generate meaningful **business insights** that can help improve decision-making in an e-commerce business.

The project demonstrates how **data analytics can be used to understand customer purchasing patterns and improve business strategy.**

---

## 🎯 Business Objectives

The main business questions addressed in this project include:

* Which **product categories generate the most revenue**?
* Which **cities generate the highest number of orders**?
* Which **sellers contribute the most to sales**?
* What are the **most commonly used payment methods**?
